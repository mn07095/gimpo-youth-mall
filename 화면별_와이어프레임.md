# 김포청년몰 화면별 와이어프레임

## 1. 메인페이지 와이어프레임

```text
[헤더]
로고 | 카테고리 | 검색 | 로그인 | 장바구니

[메인 히어로]
큰 배너 이미지
문구: 김포 청년이 고른 신선한 과일
버튼: 제철 과일 보기 / 베스트 상품 보기

[카테고리 바로가기]
사과 | 배 | 감귤 | 토마토 | 제철과일 | 선물세트

[베스트 상품]
상품카드 4~8개

[오늘의 제철 과일]
가로 카드형 상품 리스트

[청년 상인 소개]
이미지 + 소개 문구 + 자세히 보기

[김포청년몰의 약속]
산지 직송 | 신선 배송 | 품질 검수 | 믿을 수 있는 결제

[공지 / 고객안내]
배송 안내 배너 또는 공지사항

[푸터]
사업자 정보 | 고객센터 | 이용약관 | 개인정보처리방침
```

메인페이지 포인트:

- 첫 화면에서 "김포청년몰"의 정체성이 바로 보여야 한다.
- 단순한 상품 진열보다 브랜드 소개와 신뢰 요소가 함께 보여야 한다.
- 메인 배너 아래에서 곧바로 상품 탐색이 가능해야 한다.

## 2. 상품목록 페이지 와이어프레임

```text
[헤더]

[카테고리 타이틀]
예: 사과
설명: 아삭하고 신선한 제철 사과를 만나보세요

[필터 / 정렬 바]
정렬: 인기순 | 최신순 | 낮은 가격순 | 높은 가격순
필터: 할인상품 | 무료배송 | 품절 제외

[상품 리스트]
+-------------------+
| 상품이미지         |
| 상품명            |
| 원산지            |
| 할인율 / 판매가   |
| 리뷰 수 / 찜 버튼 |
+-------------------+

+-------------------+
| 상품이미지         |
| 상품명            |
| 원산지            |
| 할인율 / 판매가   |
| 리뷰 수 / 찜 버튼 |
+-------------------+

[페이지네이션 또는 더보기]

[푸터]
```

목록페이지 포인트:

- 모바일에서는 2열 또는 1열 카드 구조로 자연스럽게 바뀌어야 한다.
- 카드에서 가격, 할인율, 이미지가 가장 먼저 보여야 한다.
- 클릭 시 상품상세 페이지로 진입하는 흐름이 명확해야 한다.

## 3. 상품상세 페이지 와이어프레임

```text
[헤더]

[상단 상세 영역]
좌측: 상품 대표 이미지 / 썸네일
우측: 상품 정보

상품명
별점 / 리뷰 수
정가
할인가
배송정보
원산지 / 출고일 / 보관방법

[옵션 선택 박스]
옵션 선택 드롭다운
수량 선택
선택 상품 요약
총 결제금액

[구매 버튼]
찜하기 | 장바구니 | 바로구매

[추천 상품 간단 노출]

[상세 탭]
상세정보 | 리뷰 | Q&A | 배송/환불 안내

[상세정보 본문]
상세 이미지
상품 설명
원산지 / 중량 / 특징 / 수확시기

[리뷰 영역]
평점 요약
리뷰 리스트

[Q&A 영역]
질문 리스트
질문 작성 버튼

[배송 / 교환 / 환불 안내]

[연관상품]

[푸터]
```

상세페이지 포인트:

- 구매 버튼은 항상 눈에 잘 띄어야 한다.
- 옵션 선택과 가격 계산은 복잡하지 않게 직관적으로 보여야 한다.
- 예시 사이트처럼 정보량은 충분하되 더 따뜻한 분위기로 재해석해야 한다.

## 4. 장바구니 페이지 와이어프레임

```text
[헤더]

[장바구니 제목]

[장바구니 상품 리스트]
체크박스 | 상품이미지 | 상품명 | 옵션 | 수량 | 가격 | 삭제

[주문 요약 박스]
상품금액
+ 배송비
- 할인금액
= 최종 결제금액

[버튼]
선택상품 삭제 | 계속 쇼핑하기 | 주문하기

[푸터]
```

장바구니 포인트:

- 수량 수정과 삭제가 간단해야 한다.
- 금액 계산이 한눈에 보여야 한다.
- 모바일에서는 주문 요약 박스를 하단 고정형으로 고려할 수 있다.

## 5. 주문 / 결제 페이지 와이어프레임

```text
[헤더]

[주문자 정보]
이름 / 연락처 / 이메일

[배송 정보]
수령인 / 주소 / 상세주소 / 배송메모

[주문 상품 확인]
상품명 / 옵션 / 수량 / 금액

[결제 정보]
상품금액
+ 배송비
- 할인금액
= 최종 결제금액

[결제수단 선택]
카드 | 간편결제 | 무통장입금 등

[약관 동의]

[결제하기 버튼]

[푸터]
```

주문페이지 포인트:

- 입력 단계가 많아도 피로하지 않도록 구역을 명확히 나눈다.
- 최종 결제금액은 항상 눈에 띄어야 한다.
- 결제 신뢰감을 주는 문구와 안내를 함께 배치한다.

## 6. 주문완료 페이지 와이어프레임

```text
[완료 아이콘 또는 일러스트]
주문이 완료되었습니다

주문번호
결제금액
배송지 요약

[버튼]
주문내역 보기 | 계속 쇼핑하기
```

## 7. 모바일 대응 기준

- 헤더 메뉴는 햄버거 메뉴로 축소
- 메인 배너는 세로 비율에 맞게 재구성
- 상품목록 카드는 모바일 2열 기준 우선 검토
- 상세페이지 구매 영역은 하단 고정 구매바 검토
- 장바구니와 주문 페이지는 세로 스크롤 중심으로 단순화

## 8. 디자인 해석 가이드

예시 사이트에서 가져올 것은 `구매 흐름`이고, 바꿔야 할 것은 `브랜드 인상`이다.

김포청년몰 스타일로 바꾸는 방법:

- 헤더에 지역성과 브랜드 문구를 넣는다.
- 메인 배너에서 과일만 보여주지 말고 청년몰 스토리를 함께 전달한다.
- 상품상세 페이지에서도 상단 정보박스를 더 정돈되고 따뜻한 톤으로 구성한다.
- 리뷰/Q&A 탭은 유지하되 전체 레이아웃은 더 여유 있고 친근하게 설계한다.
